Mob Origins

Mob Origins is an origins addon that currently adds over 10 unique origins, each with different powers and abilities. If you are looking to use some powers from this in a datapack, you can checkout the wiki.

Missing Wilds

Missing Wilds strives to add all the content that was dropped or scrapped from the Wild Update. This includes things like fallen logs, and the birch forest changes from the concept art shown at Minecraft Live 2021.

Size Shifting Potions

Size Shifting Potions adds potions to the game which all affect your size in various fun ways, such as shrinking you down, growing you up, or even making you super wide.

Glow Banners

Allows you to make banners glow, by right clicking them with a glow ink sac.

Extra Sponges

Extra Sponges adds new types of sponges to help clear out larger areas of water. It also includes a lava sponge for lava clearing.

Loading Screen Tips

Adds useful tips to the loading screen, that can be customized through resource packs.

Smelting Touch

Adds a new enchantment, Smelting Touch, which smelts items dropped by blocks mined with the enchantment.

Lucky Ducks
